Proyecto M2

¡Bienvenidos a la presentación de nuestro emocionante proyecto, EmanuelCine! En este proyecto, hemos creado una plataforma web dedicada a la experiencia cinematográfica, diseñada con HTML, estilizada con CSS y dotada de interactividad gracias a JavaScript, sumado a la integración de APIs externas para obtener información en tiempo real sobre próximos estrenos y noticias cinematográficas!
Acompáñennos en este viaje por el fascinante mundo del cine a través de la web.

Proximos Pasos

Diseñar el Back-End: Continuaremos por crear APIs, el servidor y nuestra base de datos para incluir una variedad aún mayor de películas y talentos cinematográficos.
Mejoras de Interfaz y Experiencia de Usuario: Siempre estamos abiertos a sugerencias para mejorar la interfaz y la experiencia del usuario.
